# Calculator Project (in python)

This is a simple calculator built using Python in VS Codde.

## Features
-   Basic arithmetic operations
-   Options provided: History, Redo, Clear History, Exit, Calculate
-   Final version after improvements

## How to run

You can run this project using any terminal (VS Code terminal, Bash, CMD, or Powershell).

### Step 1: Open terminal in the project folder

### Step 2: Run the program

python main.py
